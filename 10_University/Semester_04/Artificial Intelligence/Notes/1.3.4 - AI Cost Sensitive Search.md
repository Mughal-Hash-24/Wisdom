# Deep Dive: Cost-Sensitive Search

## 1. The Paradigm Shift
Uninformed search (BFS/DFS) assumes all steps are equal (Cost = 1). In the real world, roads have different lengths, and actions have different energy costs.
**Cost-Sensitive Search** (specifically Uniform Cost Search) explores the "cheapest" path, not just the "shallowest" one.

## 2. Cost: Goal vs. Distance
*   **Step Cost ($c(n, a, n')$):** The cost to move from Node $A$ to Node $B$. (e.g., 5km).
*   **Path Cost ($g(n)$):** The cumulative cost from Start to current Node $n$.
    *   $g(n) = \sum \text{step costs}$.
*   **Goal Cost:** We want to minimize $g(Goal)$.

## 3. The Algorithm (UCS)
UCS is essentially BFS, but instead of a FIFO Queue, it uses a **Priority Queue** ordered by $g(n)$.
*   **Logic:** Always expand the node with the *lowest cumulative cost* found so far.

## 4. Is it Greedy?
Strictly speaking, UCS is **not** "Greedy Search" (which uses heuristics). However, UCS is locally "greedy" in terms of **past cost**. It greedily consumes the cheapest known path frontier.
*   *Note:* In AI terminology, "Greedy Best-First Search" refers to using $h(n)$ (future estimate), while UCS uses $g(n)$ (past history).

## 5. Visual Example: The Trap
**Graph:**
*   Start -> A (Cost 1)
*   Start -> B (Cost 10)
*   A -> Goal (Cost 100)
*   B -> Goal (Cost 5)

**BFS Execution:**
1.  Expands Start. Sees A and B.
2.  Expands A. Sees Goal. **Stops.**
3.  **Result:** Path Start -> A -> Goal. **Total Cost: 101.** (Sub-optimal).

**UCS Execution:**
1.  Priority Queue: `[(Start, 0)]`
2.  Pop Start. Add Neighbors: `[(A, 1), (B, 10)]`.
3.  Pop A (Lowest cost 1). Add Neighbor: `[(B, 10), (Goal, 101)]`.
4.  Pop B (Lowest cost 10). Add Neighbor: `[(Goal_via_B, 15), (Goal_via_A, 101)]`.
5.  Pop Goal_via_B (Lowest cost 15).
6.  **Result:** Path Start -> B -> Goal. **Total Cost: 15.** (Optimal).

## 6. Analysis
*   **Optimality:** Yes, guaranteed.
*   **Completeness:** Yes (if costs > 0).
*   **Complexity:** $O(b^{C^*/\epsilon})$ where $C^*$ is optimal cost and $\epsilon$ is min step cost.
